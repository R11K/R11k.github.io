Description

Farmtron is a CoreXY CNC-style robot built on a gantry system with an automatic tool changer, letting it swap between tools mid-sequence to seed, water, monitor, and tend a plot without manual intervention. Building it demanded motion-control skills I taught myself along the way — coordinating the CoreXY kinematics, the tool-change docking, and the sequencing logic into one reliable machine.

It represents a design-confidence breakthrough for me: taking a complex multi-subsystem build from concept through a working, repeatable prototype.
